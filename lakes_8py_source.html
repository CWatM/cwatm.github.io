126  # Now some assumption to make life easier:
127  # 1.) storage volume is increase proportional to elevation: S = A * H
128  # H: elevation, A: area of lake
129  # 2.) outgoing discharge = c * b * H **2.0 (c: weir constant, b: width)
130  # 2.0 because it fits to a parabolic cross section see Aigner 2008
131  # (and it is much easier to calculate (that's the main reason)
132  # c for a perfect weir with mu=0.577 and Poleni: 2/3 mu * sqrt(2*g) = 1.7
133  # c for a parabolic weir: around 1.8
134  # because it is a imperfect weir: C = c* 0.85 = 1.5
135  # results in a formular : Q = 1.5 * b * H ** 2 = a*H**2 -> H =
